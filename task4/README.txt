Выполнил: Ростислав Пеховский

Для обработки XML файла был разработан ряд классов,
отвечающих за:
1. Валидацию (MedicineValidator с помощью XSD схемы)
2. Разбор XML файла (handler package)
3. Построения объектов на основе данных, полученых при разборе (builder package)
4. Конструкторы: отвечают за выполнение всех действий и возврат
полученного результата (constructor package)

Для вывода данных с помошью веб-страницы:
1. Cервлет (MainPageServlet)
2. Сервис класс для получения данных с файла (MedicineService)
3. JSP страница для отображения данных и для вывода ошибок

Также для настроек приложения был создан класс AppProperty, которых загружает
параметры приложения (такие как пути до файлов и паттерн даты) с ресурсов
(файл application.properties).
